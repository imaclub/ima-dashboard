# IMA Dashboard — Tour de Contrôle

Back-office de pilotage pour le restaurant IMA (Val d'Isère).

## Modules
- Dashboard principal (CA, KPIs, alertes)
- Finance & P&L (Pennylane)
- Food & Bar Cost
- Staff & RH (équipe + tips)
- Stocks & Inventaire (avec analyse écarts Trivec)
- CRM & Réservations (SevenRooms)
- Caisses (Trivec)
- Administration multi-établissement

## Déploiement
Déployé sur Vercel → dashboard.ima-club.com
